# Tarea: Cálculo de Área en Python
Este programa calcula el área de un triángulo utilizando diferentes tipos de datos.

## Características técnicas:
* **Identificadores:** Cumplen con la regla de iniciar con letra.
* **Nomenclatura:** Funciones escritas en `snake_case`.
* **Tipos de datos:** Uso de `string`, `float`, `int` y `boolean`.
